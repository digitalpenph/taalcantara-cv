===========
ABOUT Theme
=========== 
Workflow is a clean and fully responsive blogging theme with an optional home page slider to feature your best posts. It has a range of out of the box customizer options for the logo and header layout, favicons, colors, author bios and social media icons. Author pics are powered by Gravatar so you'll want to make sure all authors set up a Gravatar account associated with their site email address.

This theme is compatible with WordPress Version 3.8 and above and it supports the new theme customization API (https://codex.wordpress.org/Theme_Customization_API).

Supported browsers: Firefox, Opera, Chrome, Safari and IE9+ (Some css3 styles like shadows and rounder corners are not supported by IE8 and below).
